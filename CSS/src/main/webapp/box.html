box-model

  1. inline - 인라인요소. 한 줄에 나란히 옴.
    상하 margin 적용 불가, 크기 적용 불가.
  2. block - 블럭요소. 한 줄에 나란히 올 수 없음.
    margin, 크기 적용 가능.

display 요소의 성격 지정

  • inline block인 요소를 inline으로 바꿈
  • block inline인 요소를 block으로 바꿈
  • inline-block inline과 block의 특징을 모두 가진 요소로 바꿈
    (한 줄에 나란히 오면서 크기와 margin 적용 가능)
  • none 화면에서 요소가 사라짐
