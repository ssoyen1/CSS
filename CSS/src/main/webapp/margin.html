margin

테두리(border) 기준 요소의 외부에 적용되는 여백

margin의 방향별 지정법

  • margin-top:n; 위에만 적용
  • margin-right:n; 우측에만 적용
  • margin-bottom:n; 아래만 적용
  • margin-left:n; 좌측에만 적용

margin의 값 갯수별 지정법

  • margin:n; 양사방 동일값 적용
  • margin:n n; 상하, 좌우 순으로 적용
  • margin:n n n; 상, 좌우, 하 순으로 적용
  • margin : n n n n ; 상, 우, 하, 좌 순으로 적용

inline요소들의 margin 적용시

상하는 적용불가, 좌우는 인접요소와 값이 중복적용

bold (margin: 2em;) bold (margin: 3em 1em;) bold (margin-left: 1em;)

[margin: n auto;] 항상 정중앙에 위치하는 요소 margin-left: auto; margin-right: auto; 로 좌우 margin을 항상 같게 하는 auto적용
