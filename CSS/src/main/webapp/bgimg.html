background 배경

  1. background-color:컬러값;배경 색상
  2. background-image:url("이미지경로"); 배경 이미지

background-image 2개 적용하기

background-image: url("이미지경로1") , url("이미지경로2");
(기타 세부 속성들도 ,로 각각 추가 가능) <span>은 문자열을 묶을 때 사용.

background-image의 세부 속성들

  • background-repeat: ; 반복 지정
    • repeat 기본값. 양사방 반복
    • repeat-x 가로로만 반복
    • repeat-y 세로로만 반복
    • no-repeat 반복 안함
  • background-size: ; 크기 지정
    • cover 영역을 이미지 하나로 전체 다덮음
    • contain 이미지 하나를 온전히 보이게 영역 덮음
    • n n 가로 세로 순으로 직접 지정
  • background-position: ; 위치 지정
    • left/center/right top/center/bottom 조합
      기본값 left top (0 0)
    • n n 가로 세로 순으로 직접 지정
  • background-attachment: ; 스크롤여부 지정
    • scroll 기본값. 브라우저스크롤바에는 스크롤, 자신 영역스크롤바에는 고정
    • fixed 브라우저스크롤바와 자신 영역스크롤바 모두에 고정
    • local 브라우저스크롤바와 자신 영역스크롤바 모두에 스크롤
