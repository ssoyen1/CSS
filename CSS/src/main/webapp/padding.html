padding

테두리(border) 기준 요소의 내부에 적용되는 여백

padding의 방향별 지정법

  • padding-top:n; 위에만 적용
  • padding-right:n; 우측에만 적용
  • padding-bottom:n; 아래만 적용
  • padding-left:n; 좌측에만 적용

padding의 값 갯수별 지정법

  • padding:n; 양사방 동일값 적용
  • padding:n n; 상하, 좌우 순으로 적용
  • padding:n n n; 상, 좌우, 하 순으로 적용
  • padding : n n n n ; 상, 우, 하, 좌 순으로 적용
