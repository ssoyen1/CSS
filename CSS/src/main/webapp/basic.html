css basic

css 문법 선택자 {속성:속성값;}

css 적용방식 3가지

  • External CSS - 스타일소스를 외부파일(abc.css)에 따로 만들고 웹페이지와 연결하여 적용.
    (head안에 <link href="abc.css" rel="stylesheet">)
  • Internal CSS - 스타일소스를 웹페이지의 head안에 style태그로 적용. (<style type="text/css"> ... </style>)
    해당 페이지에만 스타일 적용됨.
  • Inline CSS - 스타일소스를 body안의 해당 태그 뒤에 (tag style="속성:속성값;") 바로 적용.
    소스가 길어지고 통일된 스타일 적용이 어렵고 유지보수 불리함.
    CSS적용 우선 순위가 가장 높음, body만 수정가능한 상황에 적용.

css 대표 선택자 3가지

  • tag선택자 - tag그대로 선택자로 활용. 강제, 자동 적용
  • id선택자 - tag에 id를 추가하고 #id형식으로 활용. 스타일 내용을 한번만 활용 가능.
    구조적인 목적
  • class선택자 - tag에 class를 추가하고 .class형식으로 활용. 스타일 내용을 중복, 조합 적용 가능.
    디자인적인 목적
