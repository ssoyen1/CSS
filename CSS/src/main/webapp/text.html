text- 로 시작하는 글자속성들

  • text-align: ; 글자 수평 정렬 (영역내의 내용이 ~로 정렬 됨)
    1. left 기본값. 왼쪽 정렬
    2. center 중앙 정렬
    3. right 우측 정렬
    4. justify 양끝정렬
  • text-decoration: ; 글자의 줄들
    1. overline 글자의 윗줄
    2. line-through 글자의 가로지르는 줄(취소선)
    3. underline 글자의 밑줄
    4. none 링크 걸린 글자의 밑줄 제거
  • text-indent: ; 글자 들여쓰기
  • text-transform: ; (영문)글자 대소문자 변형
  • text-shadow: ; 글자의 그림자
  • text-overflow: ; 영역 밖의 글자 처리
